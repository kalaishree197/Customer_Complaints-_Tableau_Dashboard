
# 📊 Customer Complaints Dashboard - Tableau Analysis

## *Interactive Tableau Dashboard for Consumer Complaint Analysis*

A comprehensive data visualization project analyzing U.S. consumer complaints data using **Tableau**. The dashboard provides deep insights into complaint trends, company performance, product categories, and geographic distribution.

## 📋 Project Overview

This project demonstrates end-to-end business intelligence and data visualization:

- Data connection and blending from multiple sources (complaints data + state mapping)
- Creation of interactive visualizations and a unified dashboard in Tableau
- Analysis of complaint volume, resolution rates, timely responses, and regional patterns
- Built with advanced Tableau features: calculated fields, parameters, filters, and maps

**Project Type**: Tableau Dashboard  
**Based on**: Consumer Financial Protection Bureau (CFPB) Consumer Complaint Database

## 🛠 Tech Stack

- **Tableau Desktop** – Dashboard development, calculated fields, parameters, dual-axis charts, maps

  - `complaints_Full Data.csv`

## 🧠 Key Business Insights

- Dominant Complaint Categories
Most complaints cluster around Bank Services, Credit Cards, and Checking/Savings Accounts. Mortgages form a smaller but notable share. Product grouping in the dashboard highlights these as the core areas of consumer friction.
- Geographic Hotspots
The state map (with tile chart option) reveals clear regional patterns in complaint volume. Some states show significantly higher complaints per capita — useful for identifying priority areas for financial institutions or regulators.
- Timely Response Performance
Companies generally perform well on timely responses, but variation exists across products and companies. The dashboard tracks "Timely response?" and company response types (e.g., explanation, relief, or no timely response). High timely rates are a positive signal, while gaps indicate improvement opportunities.
- Company Response Trends
Analysis of Company public response and Company response to consumer shows patterns in how firms handle complaints (e.g., closed with explanation vs. monetary/non-monetary relief). Certain products see higher rates of relief or disputes.
- Submission Channels & Consumer Behavior
Complaints arrive via multiple channels (Submitted via). The dashboard breaks this down, helping understand whether web, phone, or other methods dominate and correlate with resolution speed.
- Time-Based Patterns
Date Received / Date Submitted fields enable trend analysis (yearly or monthly). Complaint volume shows fluctuations, with potential spikes tied to economic events or product-specific issues.
- Issue & Sub-Issue Depth
Common issues (and sub-issues) vary by product. The dashboard allows drilling into specifics, revealing recurring pain points like account management, billing, or reporting errors.
